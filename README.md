# kotlin-material-ui
Kotlin Wrapper Library of Material-UI

## Installation

```
repositories {
    jcenter()
    // or maven { url 'https://dl.bintray.com/subroh0508/maven' }
}

dependencies {
    implementation "subroh0508.net.kotlinmaterialui:core:${version}"
}
```
